## Formato dos Slides

### Especificações recomendadas:
- **Formato**: PowerPoint (.pptx) convertido para PDF
- **Orientação**: Paisagem (16:9)
- **Cores**: Seguir identidade visual Blueberry + SESI
  - Primária: Azul #0052CC
  - Secundária: Laranja #FF6B00
  - Acentos: Verde #00875A
- **Tipografia**: Inter (títulos) e Open Sans (corpo)
- **Logos**: Blueberry Math + SESI em todas as páginas
- **Elementos visuais**: Ícones, capturas de tela, diagramas

### Estrutura sugerida:
1. **Capa**: Título do módulo + número + logos
2. **Objetivos de aprendizagem**: O que será ensinado
3. **Conteúdo principal**: Tópicos divididos em seções
4. **Exemplos práticos**: Casos de uso, screenshots
5. **Resumo**: Pontos-chave do módulo
6. **Próximos passos**: Link para próximo módulo

## Como usar no site:

### Link de download em bronze.html (para cada módulo):
```html
<div class="module-meta">
  <a href="downloads/materiais-apoio/modulo-1-slides.pdf" 
     class="btn btn-secondary btn-small" 
     download>
    <i class="fas fa-download"></i> Baixar Slides
  </a>
</div>
```

### Adicionar novos materiais:
1. Criar slides no PowerPoint seguindo especificações
2. Exportar como PDF de alta qualidade
3. Salvar na pasta `downloads/materiais-apoio/`
4. Nomear como `modulo-[numero]-slides.pdf`
5. Verificar que o link no bronze.html aponta corretamente
